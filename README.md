The AI System to Detect Spoofed Voice Calls project addresses the growing security risks as
sociated with voice cloning and deepfake audio technologies. With rapid advancements in artificial intelligence, cybercriminals can now generate highly realistic fake voices that can be used for fraud, identity theft, and impersonation attacks. Many existing communication systems lack reliable mechanisms to verify whether a voice is genuine or artificially generated, making users vulnerable to voice-based scams. This project provides an effective solution by developing an intelligent web-based application capable of detecting spoofed voice calls using machine learn
ing techniques.  
The system analyzes uploaded audio recordings and extracts key speech features such as Mel
FrequencyCepstralCoefficients(MFCC)using the Librosalibrary. These features are processed
using a RandomForestclassifier trained to distinguish between genuine and spoofed voices with
high accuracy. The web application is built with a user-friendly interface that allows users to
upload audio files and receive instant classification results.The system is deployed using a Flask
backend and modern web technologies, enabling real-time detection and accessibility across
platforms.
By leveraging artificial intelligence and audio signal processing, the proposed system enhances
digital security by identifying manipulated or AI-generated voices. The project aims to reduce
the risk of voice-based fraud and improve trust in communication systems, while offering scala
bility for future integration with telecom services, banking security systems, and fraud detection
platforms.
